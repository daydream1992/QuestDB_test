# 市场数据模块说明

## 目录结构

```
市场数据模块/
├── 市场数据查询.py              # 查询模块
├── 全市场数据架构导出.py        # 数据导出脚本（通达信环境运行）
├── 市场数据/                    # 数据文件目录
│   ├── 名称映射.json           # code → {name, type}
│   ├── 板块索引.json           # block_code → {name, categories:[{type,category}]}
│   ├── 个股板块全景.json        # stock_code → [{code, name, type}]
│   ├── 行业/
│   │   ├── 行业层级树.json     # [{code, name, children:[{code,name,children:[{code,name}]}]}]
│   │   └── 股票行业映射.json   # stock_code → {level1:{code,name}, level2:{code,name}, level3:{code,name}}
│   ├── 概念/
│   │   ├── 概念板块列表.json   # [{code, name}]
│   │   └── 概念成分股.json     # {block_code: [stock_code, ...]}
│   ├── 风格/
│   │   ├── 风格板块列表.json
│   │   └── 风格成分股.json
│   ├── 地区/
│   │   ├── 地区板块列表.json
│   │   └── 地区成分股.json
│   └── 指数/
│       ├── 指数列表.json
│       └── 指数成分股.json
└── 模块说明.md
```

## 查询 API

```python
import sys
sys.path.insert(0, r'k:\tdxdata-master\市场数据模块')
from 市场数据查询 import MarketData

m = MarketData()

# 代码 → 名称和类型
m.get_name('600519.SH')
# → {'name': '贵州茅台', 'type': '主板'}

# 批量查名称
m.batch_names(['600519.SH', '000001.SZ'])
# → [{'code': '600519.SH', 'name': '贵州茅台', 'type': '主板'}, ...]

# 股票 → 所有归属板块
m.get_stock_blocks('000001.SZ')
# → [{'code': '880609.SH', 'name': '跨境支付CIPS', 'type': '概念板块'}, ...]

# 板块 → 成分股列表
m.get_block_stocks('880506.SH')
# → ['000016.SZ', '000049.SZ', ...]

# 股票 → 行业链
m.get_stock_industry('000001.SZ')
# → {'level1': {'code': '881385.SH', 'name': '银行'}, 'level2': {...}, 'level3': {...}}

# 板块分类（多分类支持，如同时是概念和指数）
m.get_block_categories('880506.SH')
# → [{'type': '概念板块', 'category': '概念'}]

# 按名称模糊搜索板块
m.find_block_by_name('5G')
# → [{'code': '880506.SH', 'name': '5G概念', 'categories': [...]}]

# 按名称模糊搜索板块（限定分类）
m.find_block_by_name('银行', category='行业')
# → 只返回行业板块

# 按名称模糊搜索股票
m.find_stock_by_name('茅台')
# → [{'code': '600519.SH', 'name': '贵州茅台', 'type': '主板'}]

# 在指定板块中找股票（任一属于）
m.find_stocks_in_blocks(['银行', '大盘蓝筹'])
# → ['000001.SZ', '600000.SH', ...]

# 在指定板块中找股票（同时属于所有）
m.find_stocks_in_blocks(['银行', '大盘蓝筹'], match_all=True)

# 行业层级树
m.get_industry_tree()
# → [{'code': '881385.SH', 'name': '银行', 'children': [...]}]

# 板块索引（全量）
m.get_block_index()
# → {block_code: {name, categories: [...]}}

# 数据统计摘要
m.stats()
# → {'名称映射': 7967, '板块索引': 927, '股票行业映射': 5533, ...}

# 重新加载数据
m.reload()

# 切换数据目录（默认自动定位当前目录下的市场数据/）
m2 = MarketData(data_dir=r'K:\txdlianghua\PYPlugins\user\市场数据')
```

## 快捷函数

## 数据更新流程

### Step 1: 通达信环境导出
在 `K:\txdlianghua\PYPlugins\user\` 目录下运行：
```
python 全市场数据架构导出.py
```
输出到 `K:\txdlianghua\PYPlugins\user\市场数据\`

### Step 2: 同步到模块目录
将 `市场数据/` 子目录整体覆盖到 `k:\tdxdata-master\市场数据模块\市场数据\`

### Step 3: 查询模块自动感知
`市场数据查询.py` 无需修改，`MarketData()` 实例化时自动加载新数据。
如需强制刷新：`reload()`

## 注意事项

- **数据来源**：通达信 TqCenter API，需启动通达信客户端运行导出脚本
- **更新频率**：建议每周/每月定期运行导出脚本同步更新
- **懒加载**：数据在首次访问时加载，非首次访问不重新读取文件
- **板块多分类**：部分板块代码可能同时属于多个分类（如既在概念又在指数），`get_block_stocks` 按优先级行业>概念>风格>地区>指数返回
- **行业成分股**：通过反向推导获取，非独立文件，查询较慢（约O(n)）
- **搜索函数**：全量遍历，百万级数据下建议配合缓存使用
